# RoboSort Pro

## Overview
RoboSort Pro is an intelligent object sorting robot that classifies objects based on:

- Shape (Circle / Rectangle)
- Color (Black / White)
- Size (Small / Large)
- Weight (Light / Heavy)

The system uses Raspberry Pi, Computer Vision, Machine Learning, Load Cell, Robotic Arm, TFT Display and Flask Dashboard.

---

## Features

- Real-time object detection
- Shape recognition using OpenCV
- Weight measurement using HX711
- Machine Learning classification
- Automated robotic arm sorting
- TFT display output
- Admin dashboard
- Viewer dashboard
- System logging

---

## Hardware Used

- Raspberry Pi 4B
- Raspberry Pi Camera
- HX711 Load Cell Amplifier
- Load Cell
- PCA9685 Servo Driver
- SG90 Servos
- MG996R Servos
- TFT Display
- 6 DOF Robotic Arm

---

## Software Stack

- Python
- OpenCV
- TensorFlow Lite
- Flask
- SQLite
- NumPy

---

## Project Workflow

1. Capture object image
2. Detect shape
3. Detect color
4. Calculate size
5. Measure weight
6. ML classification
7. Display result
8. Sort object
