# Dataset

The training dataset used for object classification is not included in this repository due to size limitations.

Dataset classes:

1. Black Small Circle Light
2. Black Small Circle Heavy
3. Black Big Circle Light
4. Black Big Circle Heavy

5. White Small Circle Light
6. White Small Circle Heavy
7. White Big Circle Light
8. White Big Circle Heavy

9. Black Small Rectangle Light
10. Black Small Rectangle Heavy
11. Black Big Rectangle Light
12. Black Big Rectangle Heavy

13. White Small Rectangle Light
14. White Small Rectangle Heavy
15. White Big Rectangle Light
16. White Big Rectangle Heavy

Each object image is labeled according to:
- Shape: Circle / Rectangle
- Color: Black / White
- Size: Small / Big
- Weight: Light / Heavy

The dataset was used to train the machine learning model responsible for object classification in RoboSort Pro.
